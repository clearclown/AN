## 仕様書フォーマット

各仕様書は以下のセクションを含みます：

1. **概要** - 機能の目的と概要
2. **ユースケース** - 想定される使用シナリオ
3. **入力** - コマンド引数・オプション
4. **出力** - 正常時・異常時の出力
5. **状態遷移** - ステートマシン図へのリンク
6. **エラーケース** - エラーコードと説明
7. **テストケース一覧** - TDD用テストケース
